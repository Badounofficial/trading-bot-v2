# WEEKLY_RECAPS — Saturday Recap archive

Recurring weekly recaps of Trading Bot V2 status. Format prescribed in
`OPERATOR_METHODOLOGY.md` Section III. Generated by
`scripts/generate_saturday_recap.py`.

## Cadence

- **Every Saturday at 12:00 UTC** (≈ 05:00 PDT / 04:00 PST on Badoun's Mac)
- Telegram TL;DR sent automatically to Badoun's chat
- Markdown file saved as `YYYY-MM-DD_recap.md` where YYYY-MM-DD is the
  Saturday end-of-window date

## Install on Mac

The launchd plist is included in the project at
`scripts/com.sebastien.v2.saturday_recap.plist`. Install it once :

```bash
cp ~/Desktop/trading-bot-v2/scripts/com.sebastien.v2.saturday_recap.plist \
   ~/Library/LaunchAgents/

launchctl load -w ~/Library/LaunchAgents/com.sebastien.v2.saturday_recap.plist

# Verify
launchctl list | grep saturday_recap
```

To unload :
```bash
launchctl unload -w ~/Library/LaunchAgents/com.sebastien.v2.saturday_recap.plist
```

To force-run now (test) :
```bash
launchctl start com.sebastien.v2.saturday_recap
```

Logs at `/tmp/v2_saturday_recap.out` and `/tmp/v2_saturday_recap.err`.

## Manual generation (any time)

```bash
cd ~/Desktop/trading-bot-v2

# This week's Saturday
python3 scripts/generate_saturday_recap.py

# A specific past Saturday (backfill)
python3 scripts/generate_saturday_recap.py --week-ending 2026-05-31

# Dry-run (no Telegram notification)
python3 scripts/generate_saturday_recap.py --no-telegram
```

## What the recap contains (structure)

Section by section, automatic where possible, `_to fill_` where subjective :

1. **TL;DR** — auto (commits, trades, daemon health)
2. **Memorable phrase of the week** — manual (per Operator Methodology pattern #6)
3. **Belief State Snapshot** — 5 dimensions, manual deltas vs last week
4. **Engineering Activity** — auto (git log + diff stats)
5. **Daemon Health** — auto (heartbeat, cycle, PnL, errors)
6. **Paper Trading Window** — auto (trades + funding within the 7-day window)
7. **Layered Inquiry — Surprise of the Week** — manual (per principle P9)
8. **Compression Check** — manual (per principle P7)
9. **Open Questions Pending** — reference to `STRATEGIC_LOGIC_DOC.md` Q1–Q25
10. **Next Week — Intent** — manual single sentence

The five **Belief State dimensions** tracked for V2 are :
- Methodological discipline
- Empirical validation
- Technical maturity
- Commercial viability
- Cross-asset robustness

## DST note

launchd uses local Mac time. Schedule is "Saturday 05:05 local". This
fires at 12:05 UTC during PDT (Mar–Nov) and at 13:05 UTC during PST
(Nov–Mar). The 1-hour drift across DST changes is acceptable for a
weekly cadence.
